Para la ejecución correcta de este programa, es necesario instalar los siguientes paquetes con el gestor de paquetes pip:

pip install pandas
pip install pandas_profiling 
pip install sklearn
pip install matplotlib
pip install seaborn 
pip install voila

Para ejecutar la interfáz gráfica de usuario, debe ingresar el siguiente comando en CMD, en el PATH en donde se encuentre el proyecto water_classification.ipynb alojado:

PATH>voila water_classification.ipynb
